## 产品愿景

通过 AI 学校仿真平台研究学生身心发展与职业匹配规律，最终落地为学生陪伴机器人产品。

```
Phase 1: AI School 仿真平台（研究工具）
Phase 2: 学生陪伴机器人（面向用户的产品）
```

## Phase 1 模块说明

### M1. 学生 Agent 系统

> 核心：每个学生是一个独立的 LLM Agent，具备差异化的人格和目标

| 子模块 | 职责 | 关键研究依据 |
|--------|------|-------------|
| **M1.1 人格初始化引擎** | 用户设定 MBTI 类型（4 维连续分数），驱动 Agent 行为差异 | MBTI-in-Thoughts (arXiv:2509.04343)；混合架构方案 |
| **M1.2 职业志向模型** | 设定理想职业，影响学习偏好与社交选择 | 待研究：职业心理学 × Agent 目标系统 |
| **M1.3 认知与行为框架** | Agent 的感知-思考-行动循环，人格如何调制决策 | Generative Agents 三公理：行为=f(记忆,情境,目标) |

### M2. 学校世界系统

> 核心：提供学生 Agent 活动的结构化环境

| 子模块 | 职责 | 关键考量 |
|--------|------|---------|
| **M2.1 课程与学习活动** | 模拟不同学科的学习过程，产生学业反馈 | 课程难度、学科类型、学习风格适配 |
| **M2.2 社交与校园活动** | 同伴关系、社团、竞赛等社会互动场景 | NetworkGames 研究：网络拓扑 × 人格分布的涌现效应 |
| **M2.3 环境与时间系统** | 时间推进、学期/学年节奏、校园空间 | Smallville 虚拟小镇的时空系统参考 |

### M3. 演化引擎

> 核心：驱动学校世界的自主运转，同时支持人为干预

| 子模块 | 职责 | 关键考量 |
|--------|------|---------|
| **M3.1 自主演化循环** | Agent 群体的日常自主行为、社交互动、学习进展 | 涌现行为研究：个性分化、合作涌现、社交纽带形成 |
| **M3.2 用户干预机制** | 用户可在演化过程中介入（调整环境、触发事件、引导学生） | 干预粒度设计：全局 vs 个体，直接 vs 间接 |
| **M3.3 事件与冲突生成** | 自动生成推动发展的关键事件（考试、友谊冲突、机遇等） | 叙事驱动 vs 随机生成 的平衡 |

### M4. 记忆与成长系统

> 核心：Agent 的长期记忆与身心变化的持久化

| 子模块 | 职责 | 关键研究依据 |
|--------|------|-------------|
| **M4.1 多层记忆架构** | 感知记忆→短期→长期→语义的分层存储与检索 | 记忆演进：扁平流→Zettelkasten→四层逻辑网络→生成式潜在记忆 |
| **M4.2 身心发展追踪** | 记录心理状态、社交能力、学业能力的动态变化 | LIFELONG-SOTOPIA：长期一致性衰减问题 |
| **M4.3 人格动态演变** | 人格特质随经历缓慢演变（非固定不变） | 人格一致性验证 (PersonaGym)；对齐陷阱规避 |

### M5. 研究与分析平台

> 核心：从仿真数据中提取研究洞察

| 子模块 | 职责 | 关键考量 |
|--------|------|---------|
| **M5.1 发展轨迹可视化** | 展示每个学生的身心发展时间线、关键转折点 | 数据可视化、时间线回放 |
| **M5.2 职业匹配分析** | 基于演化结果评估学生与理想职业的匹配度，推荐适合职业 | 职业适配模型、人格-职业关联 |
| **M5.3 实验对照与数据导出** | 支持多次仿真对比、参数调整、结果导出 | 可复现性、统计分析支持 |

| 子模块 | 职责 | 前置依赖 |

## 模块依赖关系

```
M1 (Agent) ──→ M3 (演化引擎) ──→ M5 (研究分析)
    ↓                ↑                    ↓
M2 (世界) ──────────┘               M6 (陪伴机器人)
    ↓                                    ↑
M4 (记忆/成长) ─────────────────────────┘
```

**建议开发优先级：**

| 优先级 | 模块 | 理由 |
|--------|------|------|
| P0 | M1.1 人格初始化 + M1.3 认知框架 | 一切的基础，且研究基础最充分 |
| P0 | M4.1 多层记忆架构 | 长期演化的核心基础设施 |
| P1 | M2.1 课程系统 + M2.2 社交系统 | 提供 Agent 活动的最小环境 |
| P1 | M3.1 自主演化循环 | 最小可运行的仿真回路 |
| P2 | M3.2 用户干预 + M3.3 事件生成 | 丰富仿真的可控性与戏剧性 |
| P2 | M4.2 身心追踪 + M4.3 人格演变 | 研究价值的核心产出 |
| P3 | M5 研究分析平台 | 数据足够后再建分析能力 |
| P4 | M6 陪伴机器人 | Phase 1 研究验证后启动 |
